# The MLflow Fable Book

**21 stories that explain every MLflow concept through everyday analogies.**

For developers, teams, and anyone new to MLflow. Available in English and Chinese.

## What's Inside

| Category | Fables |
|----------|--------|
| **Experiment Tracking** | The Chef's Kitchen, The Baker's Notebook, The Museum Vault, The Autopilot |
| **Model Management** | The Universal Adapter, The Apprentice's Logbook, The Royal Library, The Launchpad |
| **LLM & GenAI** | The Detective's Thread, The Poet's Drafts, The Harbor Master, The Test Kitchen, The Judge's Tournament, The Quality Inspector |
| **Infrastructure** | The Apartment Building, The Gatekeepers, The Signal Flares, The Blueprint |
| **Coming Soon (RFCs)** | The Tool Shed Registry, The Cold Storage, The Handshake |

Each fable has:
- **A story** with a named character and a moment where something goes wrong
- **A concept explanation** in plain terms
- **A frontend developer reference** mapping the concept to React components and data shapes

## Quick Start

```bash
npm install
npm run dev      # Start dev server → http://localhost:5173/
npm run build    # Build for production
npm run preview  # Preview production build
```

## Languages

- English (`/en/`)
- 中文 (`/zh/`)

Switch languages using the toggle in the top-right corner. More languages welcome via PR.

## Keeping Up with MLflow

The fables track the MLflow source code. See `SYNC_STATUS.md` for the last reviewed version and `CLAUDE.md` for the update workflow.
